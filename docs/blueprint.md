# **App Name**: EduWisdom AI

## Core Features:

- Secure User Access: User authentication (login/logout), personal information management, and application settings accessible via a user profile menu.
- Dynamic Knowledge Base Organization: Create, browse, and manage knowledge bases with a persistent left-hand sidebar displaying a hierarchical tree structure for institutional, shared, and personal knowledge bases.
- Contextual Knowledge Selection: Enable users to select specific folders or documents within their knowledge bases via checkboxes, which then serve as context for AI interactions, indicated by a real-time 'X items selected' counter.
- Intuitive AI Chat Interface: A responsive and interactive chat interface for AI conversations, featuring a floating text input field, voice input, upload capabilities, and a clear display of past dialogue.
- Intelligent AI Q&A and Content Tool: Provide AI-driven answers based on selected knowledge base context, along with shortcut tools for AI writing, translation, summarization, and synopsis generation.
- Interactive Welcome Dashboard: A welcoming home page featuring a banner with flat-style illustrations, a personalized greeting, a concise system introduction, and highlight cards for core modules and features, with a prominent 'New User Guide' button.
- Knowledge Document Actions: Contextual dropdown menus for knowledge base folders and documents, offering actions such as view, rename, sharing settings, and deletion.

## Style Guidelines:

- Primary action and brand color: A vibrant 'Tech Blue' (#1677FF) for key interactive elements and branding accents.
- Background color: A very light, desaturated blue (#EFF4FC) to provide a clean and professional base for the interface.
- Accent color: A complementary 'Sky Blue' (#54D5ED) to highlight secondary actions and information, ensuring good contrast and visual interest.
- Headlines and body text font: 'Inter' (sans-serif) for its modern, neutral, and highly readable qualities suitable for an academic and technical application.
- Utilize 'Lucide React' for a consistent set of clean, scalable, and modern line-art icons, aligning with the B-side SaaS management style.
- A global, responsive layout featuring a fixed left sidebar for navigation and knowledge base management, a prominent top bar with breadcrumbs and user controls, and a main content area for interaction.
- Subtle and functional animations, including smooth transitions for UI state changes (e.g., knowledge base selection, chat flow progression) and gentle hover effects for interactive elements to enhance user experience without distraction.
